## Proje Amacı

Bu proje, Docker kullanarak modern bir **DevOps portfolio web sayfasını** container içinde çalıştırmayı amaçlamaktadır.  

Bu proje sayesinde öğrenilenler:

- Dockerfile oluşturma ve image build etme  
- Container çalıştırma ve port mapping  
- Static web deployment  
- Temel DevOps uygulama mantığı  

---

## Kullanılan Teknolojiler

- **Docker** – Uygulamayı containerize etmek için  
- **Nginx (Alpine)** – Hafif ve hızlı web sunucusu  
- **HTML / CSS** – Web sayfası tasarımı
